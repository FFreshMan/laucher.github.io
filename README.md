### 这是一个兴趣使然的作品库。
<br>
** 2048 **：一个仿2048的小游戏。<br>
**clock**：用canvas画的时钟，可以显示当前时间。<br>
**eclcok**：clcok的加强版，用电子钟的方式来展示，还会掉节操。<br>

**pre-slidershow**：简单的幻灯片实现。<br>
**utils**：对原生js的封装常用的功能。<br>
**oupload**：音乐可视化，后台用的java，数据库用的mysql<br>
